# TransVoice.Live 🎙️

**TransVoice.Live** — это высокопроизводительная CLI-утилита для потокового распознавания речи в реальном времени с использованием моделей OpenAI Whisper. Приложение оптимизировано для живой диктовки: оно автоматически очищает звук от шумов и компенсирует расстояние до микрофона.

## 🛠️ Системные требования

Для работы утилиты необходимы:
- `arecord` (входит в пакет `alsa-utils`) — для захвата звука.
- `xclip` — для работы с буфером обмена.
- `.NET 10.0 SDK` — для сборки проекта.

## 🚀 Установка и настройка

### 1. Сборка приложения
Выполните публикацию проекта в локальную папку `publish`:
```bash
dotnet publish app/TransVoice.Live.csproj -c Release -o ./app/publish
```

### 2. Создание симлинка (Глобальный доступ)
Чтобы запускать утилиту из любой папки под именем `transvoice-live`, создайте символическую ссылку:
```bash
mkdir -p ~/.local/bin
ln -sf $(pwd)/app/publish/TransVoice.Live ~/.local/bin/transvoice-live
```
*Убедитесь, что путь `~/.local/bin` добавлен в вашу переменную `$PATH`.*

### 3. Первоначальная настройка
Запустите команду настройки, чтобы указать путь к модели Whisper и выбрать язык:
```bash
transvoice-live settings
```
*Если вы запускаете настройку повторно, вы сможете пропустить этапы (параметры с галочкой ✔ будут сохранены).*

## 📖 Использование

- **transvoice-live live** — запуск режима диктовки.
- **transvoice-live settings** — изменение конфигурации.

### Управление в режиме диктовки:
- **Говорите в микрофон**: текст будет появляться на экране в реальном времени.
- **Нажмите ENTER**: чтобы остановить запись, вывести полный текст и автоматически скопировать его в буфер обмена.

## ✨ Особенности
- **Smart AGC**: Автоматическая компенсация расстояния. Если вы находитесь далеко от ноутбука, система усилит сигнал, чтобы Whisper его расслышал.
- **Adaptive VAD**: Интеллектуальное отсечение шумов (например, вентиляторов ноутбука).
- **Clipboard Integration**: Текст сразу готов к вставке после завершения фразы.
