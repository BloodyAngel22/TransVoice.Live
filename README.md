# TransVoice.Live 🎙️

**TransVoice.Live** — это высокопроизводительная CLI-утилита для потокового распознавания речи в реальном времени с использованием моделей OpenAI Whisper. Приложение оптимизировано для профессиональной диктовки: оно не только распознает голос, но и автоматически расставляет знаки препинания, очищает звук от шумов и адаптируется к любому микрофону.

## 🛠️ Системные требования

Для работы утилиты необходимы:
- `arecord` (входит в пакет `alsa-utils`) — для захвата звука.
- `xclip` — для работы с буфером обмена.
- `.NET 10.0 SDK` — для сборки проекта.

## 🚀 Установка и настройка

### 1. Сборка приложения
Выполните публикацию проекта в локальную папку `publish`:
```bash
dotnet publish app/TransVoice.Live.csproj -c Release -o ./app/publish
```

### 2. Создание симлинка (Глобальный доступ)
Чтобы запускать утилиту из любой папки под именем `transvoice-live`, создайте символическую ссылку:
```bash
mkdir -p ~/.local/bin
ln -sf $(pwd)/app/publish/TransVoice.Live ~/.local/bin/transvoice-live
```
*Убедитесь, что путь `~/.local/bin` добавлен в вашу переменную `$PATH`.*

### 3. Первоначальная настройка
Запустите команду настройки, чтобы указать путь к модели Whisper и выбрать язык:
```bash
transvoice-live settings
```
*Если вы запускаете настройку повторно, вы сможете пропустить этапы (параметры с галочкой ✔ будут сохранены).*

## 📖 Использование

- **transvoice-live live** — запуск режима диктовки.
- **transvoice-live settings** — изменение конфигурации.

### Управление в режиме диктовки:
- **Говорите в микрофон**: текст будет появляться на экране в реальном времени с метками времени.
- **Нажмите ENTER**: чтобы остановить запись. Полный, очищенный от меток времени и отформатированный текст будет выведен на экран и автоматически скопирован в буфер обмена.

## ✨ Особенности

- **Smart Punctuation & Context**: Использование специальных промптов и расширенного окна контекста позволяет Whisper корректно расставлять запятые, точки и тире в сложных предложениях.
- **Universal Adaptive VAD**: Интеллектуальная система детекции голоса на основе SNR (Signal-to-Noise Ratio). Она постоянно калибрует уровень фонового шума в реальном времени, подстраиваясь под любой микрофон и расстояние до него.
- **Noise Gate (Шумовой затвор)**: Физическое отсечение фонового гула в моменты тишины, что полностью исключает "галлюцинации" нейросети (появление случайных слов из шума).
- **Smart Joining (Умная склейка)**: Алгоритм автоматического объединения фрагментов речи в цельные абзацы. Система понимает, когда фраза является продолжением предыдущей (по союзам и регистру), и убирает лишние точки.
- **Performance Monitoring**: Вывод детальной статистики для каждого фрагмента: время обработки (`Work`) и длительность аудио (`Audio`).
- **Hybrid Gain Control**: Автоматическое усиление сигнала (AGC) позволяет комфортно диктовать текст, даже находясь на значительном удалении от микрофона.
